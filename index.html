關於我

你好！我是 Jimmy，一名熱愛像素完美設計的前端工程師。我的旅程始於對網站運作原理的好奇，如今已轉化為對撰寫乾淨、語意化 HTML 與高效 CSS 架構的堅持。我相信優秀的程式碼不僅是為了機器運作，更是為了創造令人愉悅的使用者體驗。

在過去的開發經驗中，我專注於解決跨瀏覽器相容性問題，並致力於提升網站效能與 SEO 排名。我喜歡挑戰複雜的 UI 互動設計，同時確保網站能在各種裝置上完美呈現。我不僅是代碼的撰寫者，更是設計與技術之間的橋樑。
